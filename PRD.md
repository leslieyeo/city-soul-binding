# 🌍 城市灵魂绑定 — PRD v6（评审修订版）

> 2026-02-11 | 纯前端 · 五行×星座×个性测试 · 城市匹配报告

---

## 1. 产品是什么

一个**纯前端的深度测试网页**。用户输入生日 + 完成 30-40 道题 → 系统综合五行、星座、个性维度 → 生成一份精美的「灵魂城市报告」，包含匹配城市、五行能量分布、性格解析、城市密码等。

通过小红书店铺以授权码形式售卖（¥1.99），纯前端校验。

**不做后端，不做服务器，不做数据库。**

---

## 2. 核心理念

**题量 = 可信度。**

市面上的竞品 3-6 道题就出结果，用户心里清楚"这就是随机的"。我们做 30-40 道题，用户花 10-15 分钟认真作答，天然觉得"算了这么多维度，结果肯定准"。

**四大体系融合 = 深度感。**

| 体系 | 作用 | 来源 |
|------|------|------|
| 五行 | 先天命理属性 | 生日自动计算 |
| 星座 | 星象性格底色 | 生日自动计算 |
| 个性测试 | 后天行为偏好 | 答题获取 |
| 城市匹配 | 最终输出 | 前三者综合 |

用户感知：**"这个测试同时分析了我的五行、星座和性格，比别的测试深多了。"**

## 3. 用户流程

```
输入授权码 → 输入基础信息 → 答题（30-40题，分4个篇章）→ 计算动画 → 报告页（可保存卡片）
```

### 3.1 授权码页
- 输入 8 位授权码（格式如 `SL7K-M2XP`）
- 前端 HMAC 校验
- 通过后进入测试

### 3.2 基础信息
- 姓名（用于报告展示）
- 出生日期（年/月/日）
- 出生时辰（可选，12 时辰下拉选择，含"不确定"选项）
- 性别（可选，用于文案微调）

系统自动计算并展示：
> "你的星座是 **天蝎座**（水象），五行日柱为 **丙火**"
>
> 给用户一个"这个测试很专业"的第一印象。

### 3.3 答题（核心体验）

**共 35 道题，分 4 个篇章，每篇章之间有过渡页。**

过渡页展示阶段性信息（如"五行能量已初步解析..."），既是休息也是仪式感。

---

**答题进度**：每题显示"当前篇章进度（如 3/9）" + 顶部总进度条。
**自动保存**：用 localStorage 保存答题进度，刷新/退出后可继续。

### 3.4 计算动画（5-8 秒）

分阶段展示，增强仪式感：
1. "解析五行天干地支..." → 五行符文旋转
2. "匹配星座能量频率..." → 星座符号闪烁
3. "计算个性维度向量..." → 数据流动效果
4. "扫描 20 座城市共鸣..." → 中国地图热力扫描
5. "锁定！你的灵魂之城..." → 城市名大字浮现

### 3.5 报告页

#### 卡片 1 — 城市明信片
- 城市名 + 专属插画/摄影背景
- 一句灵魂文案
- 姓名 + 日期 + 星座 + 五行属性标签

#### 卡片 2 — 五行能量图谱
- 五行百分比分布（五角雷达图）
- 主属性 + 辅属性说明
- 星座对五行的修正说明（如"天蝎座水象加持，水属性 +12%"）
- 性格关键词 × 4
- 与匹配城市的共鸣解析（150 字）

#### 卡片 3 — 城市深度报告
- 城市人格描述
- 城市密码 ×3（标志性体验）
- "你在这座城市的能量加成"
- 一句寄语

#### 卡片 4 — 城市车票（彩蛋）
- 复古车票设计
- 姓名 + 城市 + 日期 + 属性
- "永久有效"

每张卡片底部有「保存图片」按钮。

#### 隐藏城市（彩蛋，条件触发）
- 当 Top1 和 Top2 城市分差 < 5% 时显示
- "你还有一座隐藏城市——[城市名]"
- 增加惊喜感和分享话题

#### 重测入口
- 报告页底部："想重新探索？" 按钮
- 点击弹窗提示"相同答案 = 相同结果哦"，确认后清除数据重新开始

## 4. 匹配算法

### 4.1 五行计算（先天）

```
生日 → 天干地支（年柱+月柱+日柱，可选时柱）
每柱有对应五行属性
加权汇总 → 先天五行分布
```

### 4.2 星座映射

```
生日 → 星座 → 元素（火/土/风/水）→ 映射五行
作为先天五行的补充权重
```

### 4.3 答题计分

```
35 道题，每题 5 个选项分别对应 [木,火,土,金,水]
每题 +3 分给对应属性
总分归一化为百分比
```

### 4.4 最终合成

```
最终五行 = 先天五行 × 25% + 星座修正 × 15% + 答题五行 × 60%
→ [木%, 火%, 土%, 金%, 水%]

城市匹配 = 余弦相似度(用户向量, 城市向量)
取 Top 1 为主匹配城市
取 Top 2 为"隐藏城市"（当 Top1 和 Top2 分差 < 5% 时展示为彩蛋）
```

### 4.5 确定性

同样的输入 = 同样的结果。用户可以验证、可以和朋友对比。
